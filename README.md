# Determinant

This repository contains an implementation of an experiemntal determinant
tactic that uses a dvision-free determinant algorithm.

The algorithm was first described in [A simple division-free algorithm for computing determinants](https://www.sciencedirect.com/science/article/abs/pii/S0020019011002353)
by Richard S. Bird.

See [BirdCbv.lean](Determinant/Tests/BirdCbv.lean) for examples.
